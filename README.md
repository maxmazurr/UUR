# UUR — Unified University Resources

Студенческое учебное веб-приложение, объединяющее Notion (заметки), Obsidian (граф знаний) и Anki (интервальное повторение) в одном рабочем пространстве.

## Запуск

```bash
npm run dev       # Dev-сервер с HMR
npm run build     # Продакшн-сборка в dist/
npm run preview   # Превью сборки
npm run lint      # ESLint
```

## Текущее состояние

| Маршрут | Файл | Статус |
|---------|------|--------|
| `/` | `src/main.jsx` | Экран выбора дашборда (готово) |
| `/1` | `src/App.jsx` | Lapis — лендинг/промо страница (UI готов) |
| `/2` | `src/pages/StudyFlow.jsx` | StudyFlow — прототип приложения (UI готов, данные mock) |

Оба дашборда — UI-прототипы с захардкоженными данными. Реального слоя данных и бизнес-логики пока нет..
